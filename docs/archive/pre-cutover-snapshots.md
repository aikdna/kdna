# Pre-Cutover Documentation Snapshots

The files listed below were historical designs, product concepts, benchmark
plans, or governance models that conflicted with the current KDNA Core format
and loading boundary. A banner at the top of each long document did not prevent
search and retrieval systems from returning old normative fragments without
their warning.

On 2026-07-17 their bodies were removed from the current documentation surface.
The old bytes remain recoverable from Git at source commit
`4fe25132ecbebee7eb28470000e017e0f0d5f91e`. The blob IDs below identify the
exact previous contents.

These snapshots are not current specifications, policies, roadmaps, commands,
quality levels, or authoring requirements. Current readers should start with
[Start Here](../start-here.md), [Status](../status.md), and
[Core Narrative and Boundaries](../core-narrative-and-boundaries.md).
