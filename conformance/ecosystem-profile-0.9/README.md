# Historical Ecosystem Profile 0.9 Fixtures

These fixtures preserve a withdrawn cross-repository candidate and its exact
validation behavior. They exist for provenance and regression compatibility;
passing them does not make Ecosystem Profile 0.9 the current KDNA roadmap,
maturity model, product contract, or release gate.

Current product guidance starts from one explicit `.kdna` file or exact
user-approved attachment. It keeps possession, attachment, authorization,
applicability, projection, Host delivery, and outcome as separate facts.
