# Registry Fixture

The KDNA registry has been removed. KDNA Core v1 does not define a public registry, marketplace, or quality-badge system.

This directory is a legacy fixture for CLI/validator development only. It must not become a public domain asset catalog.

Rules:
- Do not add public domains here.
- Do not point registry entries to official packages.
- Domain assets must live in standalone repositories named `kdna-<domain>`.
- KDNA Core v1 uses local `.kdna` files; no registry required.
