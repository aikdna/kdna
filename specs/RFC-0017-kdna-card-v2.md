# RFC-0014: KDNA Card v2 (Archived)

**Status:** Withdrawn — retained as an archive marker

This proposal was not adopted into the current KDNA Core specification. It adds
no required KDNA Card fields, container entries, runtime behavior, or
compatibility commitment.

Use the maintained [KDNA Card documentation](../docs/KDNA_CARD_SPEC.md),
[KDNA SPEC](../SPEC.md), and [consumption runtime guide](../docs/consumption-runtime.md)
for public guidance. A future proposal must be self-contained, publicly
reproducible, and explicitly versioned before it can affect the protocol.
